#GracyBot
个性定制化简约生态QQ机器人框架（基于Python3.10+&amp；Napcat)，主打安全稳定、插件化扩展与便捷更新。核心特性：模块化架构（配置/日志/监控/插件管理分离），全局安全防护（日志脱敏、危险命令拦截、频率限制），支持自动回复、AI对接、群/私聊交互，插件独立开发无需改动核心。内置健康检查、版本对比、自动备份，适配Linux/Windows，支持/检测更新等命令对接GitHub实现平滑升级，适合个人/开发者快速搭建功能完备的QQ机器人
