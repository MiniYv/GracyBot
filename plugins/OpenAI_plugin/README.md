# OpenAI 插件

## 插件介绍
OpenAI插件是GracyBot的AI聊天助手模块，提供基于大语言模型的智能对话功能，支持多轮对话、人设切换和记忆管理。

## 功能特性
- 智能AI对话：通过OpenAI兼容的API接口提供自然语言交互
- 人设系统：支持自定义和切换不同的AI人设
- 对话记忆：自动记录和管理对话历史，支持清除记忆
- 严格触发机制：群聊中仅响应"//"开头的消息，避免误触发
- 主人专属命令：支持API配置、人设管理等高级功能

## 结构特点
- 配置持久化：API配置和人设数据保存在JSON文件中
- 模块化设计：核心功能与辅助功能分离
- 错误处理：完善的异常捕获和日志记录
- 安全性：严格的触发机制和权限控制

## 使用方法
- 群聊中使用"//"开头的消息进行AI对话
- 私聊中使用"//"开头的消息进行AI对话
- 主人专属命令（仅支持私聊）：
  - `/设置OpenAI API_KEY 模型 地址`：配置OpenAI API
  - `/新增人设 人设名称 人设内容`：添加新的AI人设
  - `/切换人设 人设名称`：切换当前使用的人设
  - `/清除记忆`：清除对话历史记录
