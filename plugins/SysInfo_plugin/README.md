# SysInfo_plugin 插件

## 插件介绍
SysInfo_plugin是GracyBot的系统信息模块，提供机器人的服务器系统信息的查询功能，帮助用户了解系统运行状态。

## 功能特性
- 硬件信息：展示CPU、内存、硬盘等硬件配置
- 系统状态：显示当前系统运行状态
- 进程信息：显示系统进程列表
- 负载监控：展示系统负载情况

## 结构特点
- 实时获取：获取最新的系统信息
- 信息整合：整合各类系统信息
- 格式化输出：以易读的格式展示信息
- 轻量级：查询过程占用资源少

## 使用方法
- 通过命令触发系统信息查询
- 系统会返回详细的系统信息
